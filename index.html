1. Stat Level Ranges by Tool Grade & Item Rank
Each item has a hidden stat level that scales its reforge bonuses. Higher item ranks yield higher level ranges. Rank 1 items have the best stat ranges.

2. Rank-Up Outcome Rates
Each tool use rolls a single outcome: Rank 3, Rank 2, or Rank 1 — directly. There is no step-by-step upgrade. The item's rank is set to whatever the roll lands on. Grade-specific tools massively increase the chance of hitting Rank 1.
