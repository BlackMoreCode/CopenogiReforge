2. Rank-Up Outcome Rates
Each tool use rolls one outcome: stay / upgrade — there is no downgrade or failure. Because of this, the effective Rank 2 → 1 rate is calculated from a smaller pool (excluding the Rank 3 weight).

Note: These rates are estimated from raw XML weighted values (3RankProb / 2RankProb / 1RankProb). The exact server implementation is unconfirmed, but this model is consistent with observed behavior.
